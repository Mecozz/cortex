# telegram

Telegram notification module for Cortex (TGRAM).

**notify**: Sends a message via the Telegram Bot API. Requires `telegram_bot_token` and `telegram_chat_id` stored in the Vault.

To configure: add `telegram_bot_token` (from @BotFather) and `telegram_chat_id` (your user or group ID) as Vault entries, then use the `telegram_send` command.
